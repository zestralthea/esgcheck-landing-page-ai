# OpenAI API OpenAPI Specification

## Description
The OpenAPI specification defines the structure and endpoints of the OpenAI API. It is available in YAML format. Users can access the most recent documented version or a manually updated version via provided URLs. Issues with the specification should be reported in the repository.

## Access
- Latest Documented Version: https://app.stainless.com/api/spec/documented/openai/openapi.documented.yml
- Latest Manually Updated Version: https://github.com/openai/openai-openapi/tree/manual_spec

## Issue Reporting
- Report problems with the OpenAPI specification by creating an issue in the openai/openai-openapi GitHub repository.
- For immediate assistance, visit: https://help.openai.com/en/articles/6614161-how-can-i-contact-support
